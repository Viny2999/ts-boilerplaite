# TS Boilerplaite

Boilerplaite Developed with NodeJS and Typescript.

## Instructions to Run Locally
There are two ways to run the application:
* Using Docker Compose:
  ```
  docker-compose up
  ``` 
* Using NPM:
  ```
  npm install
  npm start
  ``` 

## Tests
To start jest unit tests run:
```
npm test
```
